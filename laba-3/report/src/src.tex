\section{Анализ исходного решения}
\subsection{Методика и условия измерений}
Для анализа исходной версии использовались:
\begin{itemize}
    \item \texttt{gprof} --- распределение времени по функциям;
    \item \texttt{Valgrind memcheck} --- проверка ошибок и утечек памяти;
    \item \texttt{Valgrind massif} --- оценка пикового потребления памяти.
\end{itemize}

Тестовый набор формировался скриптом \texttt{bench\_gen.py}:
\begin{itemize}
    \item 200\,000 вставок;
    \item 400\,000 поисков;
    \item 100\,000 удалений.
\end{itemize}

\section{Консоль}
